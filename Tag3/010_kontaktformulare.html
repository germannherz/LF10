Kontakformulare in HTML

Formulare helfen uns dabei, Daten von Nutzern einzusammeln. Die wichtigsten Bausteine sind:

  • <form> = der Formular Rahmen (Container)
  • <label> = Beschriftung für die Eingabefelder
  • <input> = Eingabefeld (Bsp.: Name, Passwort, Email, Telefonnr.,...)
  • <textarea> = großes Textfeld (Bsp.: Nachrichten)
  • <select> und <option> = Dropdown auswahl
  • <button> = Button (Absenden/ Zurücksetzten)

1.Das einfach Kontakformular

Ein Kontaktformular besteht meistens aus Name,Email und Nachricht

for (bei <label>) + id
Verbindet Label und Feld, Klickst du auf das Label, sprint dein Cursor ins Feld.

2.Die wichtigen Form-Attribute: action und method

action=""
Wohin die Daten geschickt werden (Bsp.: eine Server-Adresse oder ein Skript)

method="GET"
Daten werden in die URL(=Uniform Resource Locator) geschrieben (gut für eine Suche, schlecht für unsere sensiblen Daten).

method="POST"
Daten werden im Hintergrund gesendet (Standard Formular).

HINWEIS: Ohne echten Server passiert beim Absenden oft nicht Sichtbares, aber HTML "baut" das Formular korrekt.

3.Input-Typen und ihre Funktionen

Der type sagt dem Browser, welche Art von Daten erwartet werden. Dadurch bekommt den passenden Tastaturen am Handy/Tablet und manchmal auch eine automatische Überprüfung.

placeholder=""
Ein Hinweistext im Feld (verschwindet beim tippen)
min/max
Grenze der Zahlen (oder ein Datum)
min = minimum (kleinster Wert)
max = maximum (größter Wert)

4.Pflichtfelder und einfache Validierung

Mit required kann man Felder verpflichtend setzen. Der Browser verhindert dann das Absenden, wenn etwas fehlt.

required
Muss ausgefüllt werden!!!

minlength
Mindestlänge für unseren Text.

maxlength
Maximalelänge für unseren Text.

5.Auswahlfelder: select, radio, checkbox

radio
Nur eine Option ist auswählbar.

select
Ist eine Auswahlliste zum ausklappen.

checkbox
Ist variable zwischen an und aus oder mehreren Auswahlmöglichkeiten.

Kontaktwunsch:


disabled
Option kann nicht ausgewählt werden.

selected
Option ist standardmäßig ausgewählt.

6.Buttons: submit, reset, button

type="submit"
Sendet das Formular ab.

type="reset"
Setzt alle Felder auf den Standardzustand zurück.

type"button"
Macht erst etwas, wenn JavaScript hinzugefügt wurde.
