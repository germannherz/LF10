Anchor-Tags (<a>) - Links in HTML

Mit dem Anchor Tags können wir Links erstellen. Links führen zu anderen Seiten/Dateien/Stelle auf der selben Seite.

1.Grundaufbau eines Links

So sieht ein einfacher Link im Code aus:

<a href="https://www.google.com/">Ich bin ein Link</a>

So sieht ein Link im Browser aus:

Das ist ein Link

2.Das Attribut hrf

href steht für Hypertext Reference. Das Attribut gibt an, wohin der Link führt Ohne href ist es kein richtiger Link.

  • href="./010_hallo.html"→ Link zu einer Datei im gleichen Ordner
  • href="#ziel"→ Link zu einer Stelle auf der selben Seite
  • href="https://www.google.com/"→ Link der zu einer anderen Website führt

3.Link in einem neuen Tab öffnen

Mit target="_blank"öffnet sich der Link in einem neuen Tab

<a href="https://www.google.com/" target="_blank" rel="noopener nopreferrer>Ich bin ein Link</a>

So sieht ein Link im Browser aus:

Das ist ein Link

noopener
Sicherheit bei target="_blank", dass die Zielseite, meine Seite nicht manipulieren kann.

noreferrer
Datenschutz, bedeutet die Zielseite sieht nicht, von welcher Seite der Klick kam.

rel
eng. relationship / dt. Beziehung

4.Email und Telefon Links

Email (mailto)

<a href="mailto:beispiel@gmail.com"> E-Mail schreiben </a>

Unser Browser öffnet das Standard E-mail Programm des Nutzers. Und erstellt uns automatisch: Empfänger (Email ist schon ausgefüllt), Betreff:leer, Text:leer.

Email schreiben

Telefon (tel)

< a href="tel: +49123456789">Jetzt anrufen</a>

Smartphone und Tablets öffnen direkt die Telefon App zum Anrufen. PCs können die Funktion nur nutzen, wenn die dafür zuständige App vorhanden ist.

Jetzt anrufen
