Listen in HTML

In diesem Abschnitt kann ich alles über verschiedene Listen nachlesen. HTML kennt:
<ol> (sortierte Liste),
<ul> (unsortierte Liste)
<dl> (beschreibende Liste).

1.Wann nutzen wir welche Liste

  • <ol>: Wenn die Rheinfolge wichtig ist (Bsp.: Rangliste,Anleitung,Rezept,...)
  • <ul>: Wenn die Rheinfolge keine Rolle spielt (Bsp.: Stichpunkte, To-Do-Liste,...)
  • <dl>: Für Begriffe mit erklärung (Bsp.: Glossar, FAQ, Wörterbuch)

2.Sortierte Liste (<ol>)

Eine sortierte Liste nummeriert die Einträge automatisch.

Beispiel (früh aufstehen):

  1. Augen öffnen
  2. Aufrichten
  3. Aufstehen

Wichtige Attribute

  • type: 1, A, a, I, i
  • start: Startwert festlegen
  • reversed: Nummerierung umkehren

Beispiel (Tiere):

  1. Hund
  2. Katze
  3. Kuh
  1. Hund
  2. Katze
  3. Kuh
  1. Hund
  2. Katze
  3. Kuh

3.Unsortierte Liste (<ul>)

Eine unsortierte Liste stellt Aufzählungen dar, bei der die Rheinfolge egal ist.

Beispiel (Städte)

  • Madrid
  • London
  • Paris

Wichtige Attribute

  • type: none, disc, square

Beispiel (Städte)

  • Madrid
  • London
  • Paris
  • Madrid
  • London
  • Paris
  • Madrid
  • London
  • Paris

4.Beschreibende Listen (<dl>)

Eine beschreibende Liste besteht aus: <dt> (Term = Begriff) und <dd> (Description = Beschreibung).

Beispiel (Begriffe in der IT):

HTML
Auszeichnungssprache zur Strukturierung der Website.

CSS
Sprache für das Desgin und Layout der Website

Mehrere Beschreibungen für einen Begriff

Dackel
Hund
Gilt als inelligent, lernfähig und stur.

5.Verschachtelte Listen

Listen können in anderen Listen eingebettet werden (Bsp.: Menüs oder Kategorien).

Beispiel (Elektro Geräte):

  • Produkt
    • Handys
      • Android
      • IOS

Anleitung: Verschachtelte Listen in HTML

  1. Erstelle zuerst eine normale Liste mit <ul> oder <ol>
  2. Lege einen Listeneintrag mit <li> an
  3. Schreibe den Text für den Haupteintrag in dieses <li>
  4. Öffne innerhalb des selben <li> eine neue Liste (<ul> oder <ol>)
  5. Lege in dieser neuen Liste die Unterpunkte mit <li> an
  6. Schließe zuerst die Unterliste und danach das äußere <li>
